## 論点（4本柱）

### 柱1：比較による劣等感（★ 最重要・感情の中心）
- みんなの投稿は「人生のハイライトだけを切り取った一部」
- 楽しい瞬間・盛れた1枚しか投稿されない
- 自分の"ふつうの毎日"と比べ → 「自分だけがつまらない」と錯覚する
- キラリ姉さんの自白で裏付ける

### 柱2：いいねと承認欲求
- 自分の価値を"他人がくれる数字"で測るクセがつく
- いいねが少ないと落ち込む → 数字に振り回される依存構造
- 投稿する側も「映えるかどうか」で行動を選ぶようになる

### 柱3：中毒・長時間化
- 「ちょっとだけ」が止まらないのは意志の弱さではなく**設計**
- ♾️ 無限スクロール：終わりがない。区切りを意図的に消してある
- 🎁 ガチャ方式（引っ張って更新＝可変報酬）：「次は面白い投稿が出るかも」でドーパミンが出る

### 柱4：FOMOと狙い撃ち通知（★ 山場のひとつ）
- ⏰ 24時間で消えるストーリーズ：「今見ないと消える」で何度も呼び戻す
- ❤️ **狙い撃ち通知**（最も強調）：
  - いいねをまとめてから通知するだけでなく
  - **ユーザーが一番スマホを開きやすい時間帯をデータで特定して**通知を送ってくる
  - "たまたま"ではなく**狙い撃ち**されている怖さ

| # | ヒント | 補足 |

### 導入（キャラ会話）
- くえり：「みんなのインスタ見てると楽しそうでキラキラしてるのに、自分だけなんかふつうで…。見るたびモヤモヤするし、気づいたら1時間スクロールしてた。なんでだろう…」
- でい太：「わかるよ。ぼくもYouTubeで経験した。何か理由があるはずだよ。」
- ポン太：「オレのインスタはどんぐりの写真ばっかり。」

### 🎯 このレッスンでわかること
- ✓ なぜインスタを見るとモヤモヤするか、しくみがわかる
- ✓ インフルエンサーとSNSの「本音」がわかる
- ✓ SNSが止められないように作られているしくみがわかる
- ✓ 罠から抜け出すヒントがわかる

### 場面1：「なんで私だけ…？」（比較の罠）
- くえりの悩み → キラリ姉さん登場・自白
- 「わたしの投稿、ぜんぶ"いちばんいい瞬間"だけ選んでるの。お部屋が散らかってる写真なんて載せないでしょ？」
- ❓考えてみよう：友だちの楽しそうな投稿を見てモヤッとしたこと、ある？

### 場面2：「いいね」がほしくてたまらない（承認欲求）
- キラリ姉さん：いいねの数が少ないと落ち込む、いいねのために生活が変わる
- 「100いいねつくと嬉しいけど、50しかつかないと『あれ、私の投稿つまらなかったかな』って不安になるの」

### 場面3：なぜ止められない？（中毒の設計）
- キラリ姉さん / くえり対話
- ♾️ 無限スクロール解説
- 🎁 ガチャ方式の解説（「引っ張って更新＝ガチャを引くのと同じ」）
- 深みにはまるイメージ図（YouTube版と同構造の漏斗型図）

### 場面4：あなたの時間を狙っている（FOMOと狙い撃ち通知）★山場
- ⏰ 24時間ストーリーズ
- ❤️ 狙い撃ち通知を強調（キラリ姉さんの自白）
  - 「SNSは、あなたが一番スマホを開きやすい時間を知っている。その時間を狙って通知を送ってくるの」
- でい太：「"たまたま"じゃなくて、狙われてるのか…！」

### 場面5：罠を知れば、自分で選べる（抜け出すヒント）
- くえり → ものしり仙人の解説
- 6つのヒントをカード形式で
- 締め：くえりの成長セリフ（横の比較→縦の自分軸）
  - でい太も一言共感
  - ポン太のひとこと

### 🖊 やってみよう
- Q1（ラジオ3択）：「インスタの投稿が"キラキラ"して見えるのはなぜ？」
  - ① みんなの生活が本当にキラキラしているから
  - ② 楽しい瞬間だけを選んで載せているから ← 正解
  - ③ カメラの性能がいいから
- Q2（ラジオ3択）：「SNSがあなたの"一番スマホを開きやすい時間"に通知を送るのはなぜ？」
  - ① 便利なタイミングで届けてくれる親切な機能だから
  - ② できるだけ何度もアプリを開かせるため ← 正解
  - ③ 電波が一番つながりやすい時間だから
- Q3（自由記述）：「インスタを見てモヤモヤしたこと、ある？それはどんなとき？」→ こたえのれいを見る
- Q4（自由記述）：「罠から抜け出すために、明日からじぶんでやってみることを1つ書こう」→ こたえのれいを見る
