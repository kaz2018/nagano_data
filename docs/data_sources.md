# データソース一覧

## 使用データ：長野県毎月人口異動調査

- **提供元**: 長野県公式ホームページ
- **ページ**: https://www.pref.nagano.lg.jp/tokei/tokei/jinko/jinko.html
- **調査概要**: 住民基本台帳をもとに毎月の出生・死亡・転入・転出を集計し、市町村別の人口を推計したもの
- **使用データ**: 各年1月1日現在の市町村別人口（総人口・男・女・世帯数）

### 加工内容

- 元Excelから市町村名・総人口・男・女を抽出
- 郡計・市計・県計・郡名行を除外
- 2025年ファイルの市町村順序を基準に統一
- 文字コード: UTF-8 with BOM（Excel対応）

## 長野県市町村概要

| 種別 | 数 |
|------|----|
| 市 | 19 |
| 町 | 23 |
| 村 | 35 |
| **合計** | **77** |

- 村の数（35村）は全都道府県で最多
- 県人口（2025年1月）: 1,985,513人

### 注意事項

- 住民基本台帳ベースの推計値であり、国勢調査確定値とは数値が異なる場合があります
- 2005年に飯田市へ合併された上村・南信濃村はデータに含まれません
- 東筑摩郡の麻績村・生坂村・山形村・朝日村・筑北村は個別に収録されています
