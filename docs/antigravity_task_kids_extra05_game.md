# 実装依頼：番外編5「ゲームのわな」（kids/extra05）

## 概要

子どもコースの番外編5ページ目として、スマホゲーム・ソシャゲの「やめられないしくみ」を
会話形式で教えるページを新規作成する。番外編1「YouTubeのわな」・番外編3「インスタのわな」の
続編的な位置づけだが、本ページは「わな」の告発だけで終わらせず、**ゲームのいい面**と
**自分の目標への応用（ゲーミフィケーション）**まで踏み込む点が今までの番外編と異なる。

- **タイトル**: ゲームのわな
- **サブタイトル**: 「あと1回だけ」が、なぜやめられないの？
- **テーマカラー**: インディゴ（`indigo-500` / `indigo-600`）。body背景は `bg-indigo-50`
- **登場人物**: でい太・ポン太・**カーマ博士（新キャラ・ゲームクリエイター兼研究者）** の3人のみ

### キャラクター名の由来（背景情報）

「カーマ博士」は、**John Carmack（ジョン・カーマック）** へのオマージュとして命名。
id Softwareの共同創業者で、DOOM・Quakeの3D描画エンジンを自ら書き上げた伝説的プログラマー。
「天才プログラマー」の代名詞的存在であることから、オーナーの意向でこの人物にちなんだ名前を採用した。

- 実在の人物をそのまま模したキャラクターではなく、**名前の響きのみを引用したオリジナルキャラクター**である（ヒカケンジ兄さん／キラリ姉さんと同様の距離感）。人物の容姿・経歴を再現する必要はない。
- 見た目・性格設定はオリジナルで自由に決めてよい（本ドキュメントでは「ゲームクリエイター兼研究者」「白衣に眼鏡」程度の緩い方向性のみ指定）。
- 実在人物名（Carmack / カーマック）そのものは登場させない。あくまで「カーマ博士」という独立した名前として扱うこと。

### 導入（タイトル直下のキャラ会話）

- **でい太**「最近、毎日学校から帰ったら真っ先にゲームのログインボーナスをチェックしちゃうんだ。忘れると『あー、もったいないことした』って気持ちになって。しかも、一度アプリを開くと、気づいたら1時間くらいたってる…。なんでこんなにやめられないんだろう。」
- **ポン太**「そろそろヤマルの時代かな。」
- **でい太**「そのしくみ、ちゃんとくわしい人に聞いてみたいな。カーマ博士に聞いてみよう。」

### 場面1：毎日ログインしたくなるわけ

見出し：毎日ログインしたくなるわけ

- **でい太**「カーマ博士！ 聞きたいことがあるんだ。毎日ログインボーナスをチェックしないと落ち着かない気持ちになるんだけど、これって何なの？」
- **カーマ博士**「正直に話そう。それは**わざと**そう感じるように作ってあるんじゃ。」

（仕掛けカード3枚・extra01の「ユーチューバーの工夫」パターンを流用）

- 🔥「連続ログインボーナス」：毎日開かないと記録が消えるから、休みたくても休めない気持ちにさせる
- 🏆「ランキング・フレンド機能」：友だちや知らない人と比べさせることで、負けたくない気持ちを利用する
- ⏳「終わらないクエスト・シーズンパス」：「あと少しで次のごほうび」を延々と用意して、やめる区切りをなくす

- **でい太**「YouTubeやインスタと同じだ…！ 長く遊んでもらうほど、ゲーム会社が儲かるってことだよね。」
- **カーマ博士**「うーん、半分正解で半分ちがう。広告で稼ぐタイプのゲームなら、遊んでもらう時間そのものがお金になる。じゃが、多くのゲームは**課金（アイテムやガチャにお金を払うこと）**で儲けておる。じつは、ただ長く遊んでもらうだけでは、そんなに儲からないんじゃ。」
- **でい太**「え、じゃあ何のためにログインボーナスとかランキングがあるの？」
- **カーマ博士**「毎日遊んで、どんどん夢中になってもらうためじゃ。夢中になればなるほど、『もっと強くなりたい』『友だちに負けたくない』という気持ちが大きくなる。その気持ちが大きいときほど、人は**よく考えられなくなって**、つい課金してしまう。ログインボーナスもランキングも、その気持ちを大きくするための仕掛けなんじゃよ。」
- **ポン太**「やめられない止まらない。」

まとめボックス：ログインボーナスやランキングの本当のねらいは、遊んでもらう時間を増やすことだけじゃない。夢中にさせて「もっと強くなりたい」「負けたくない」気持ちを大きくし、よく考えられなくなっているうちに課金させること。それが多くのゲーム会社の収入源になっている。

### 場面2：「あと少しで、ゴールできそう」に引っぱられるわけ

見出し：「あと少しで、ゴールできそう」に引っぱられるわけ

- **でい太**「ガチャじゃなくても、『あと少しだけ』って思っちゃう瞬間がある気がする。経験値バーがあと少しで満タンになるときとか、ダンジョンのゴールがもう少しで見えるときとか。」
- **カーマ博士**「それも、するどい発見じゃな。人は、**ゴールが近づいているのが目に見える**と、そこでやめるのが『もったいない』と感じて、もっとがんばりたくなるようにできておる。」
- **でい太**「たしかに、あと少しでレベルアップってときは、絶対そこまでやっちゃう。」
- **カーマ博士**「それに、もうひとつ大事なことがある。ゲームは、なにかをした結果が**すぐに目に見える**んじゃ。攻撃した瞬間にダメージの数字が出たり、パズルをそろえた瞬間にコンボが光ったり。**やった → すぐわかる**が何度もくり返されると、もっとやりたくなってしまうんじゃよ。」
- **でい太**「ドラクエとか、パズドラとか、にゃんこ大戦争とかも、そういう作りになってるってこと？」
- **カーマ博士**「そのとおり。ダンジョンの攻略メーターや、ステージの進み具合が見えるゲームは、みんなこのしくみを使っておる。ゴールが近くに見えて、しかも結果もすぐわかる。この2つがそろうと、人はかんたんにはやめられなくなるんじゃ。」
- **ポン太**「眠らずにゲームやりすぎると、たまに小さいおじさんが見える。」

まとめボックス：ゴールが近づいているのが目に見えると、そこでやめるのが「もったいない」と感じてがんばりたくなる。しかも、やった結果がすぐに目に見えると、もっとやりたくなる。これは、ガチャの「ランダムなごほうび」とはまた別の、やめられない理由。

### 場面3：「次こそ当たるかも」がやめられないわけ

見出し：「次こそ当たるかも」がやめられないわけ

- **でい太**「ログインしたあと、『ちょっとだけ』のつもりが、気づいたら1時間もたってるのはなぜなんだろう？」
- **カーマ博士**「それは、ゲームの中の『ごほうび』の出し方にひみつがあるんじゃ。たとえば、無料でひける宝箱や、無料ガチャは、何が当たるか**まったくのランダム**じゃろう？」
- **でい太**「うん、次はいいのが出るかもって、つい開けちゃう。」
- **カーマ博士**「そうじゃ。『次こそ当たるかも』と思うと、頭がワクワクして、何度もやりたくなってしまうんじゃ。もし当たるタイミングが最初から決まっていたら、人はそんなに夢中にならない。**いつ当たるかわからない**からこそ、やめられなくなるんじゃよ。」
- **カーマ博士**「これは、お店にある**ガチャガチャ（カプセルトイ）**や、**パチンコ**と、まったく同じしくみなんじゃ。大人がハマってしまう**ギャンブル**も、じつはこれと同じしくみでできておるんじゃよ。」
- **ポン太**「今年、雨少ないよね。」

まとめボックス：「あと1回だけ」でやめられないのは、ごほうびが当たるかどうかランダムだから。当たるタイミングがわからないからこそ、やめられなくなる。お店のガチャガチャやカードのパック、大人のパチンコも、ぜんぶ同じしくみでできている。

### 場面4：ゲームのいいところと、自分で選ぶ力

見出し：ゲームのいいところと、自分で選ぶ力

- **でい太**「じゃあゲームって、悪いことしかないの…？」
- **カーマ博士**「そんなことはない！ わしはゲームを作る仕事に誇りを持っておる。ゲームには、きみの力を育てる、いいところもたくさんあるんじゃよ。」

（いい面カード4枚・extra03の「良いところ」ボックスパターンを流用、`bg-blue-50 border-blue-200`）

- 🧠「記憶力」：マップ・アイテム・手順を頭の中で整理して覚える力が鍛えられる
- 🧩「考える力」：パズルや作戦を立てるゲームで、「次にどうするか」を考える力がつく
- 🗺️「空間認識力」：ゲームの中の世界で、自分がどこにいるか、場所をつかむ力が育つ
- 🤝「仲間と力を合わせる力」：チームプレイで役割を分担したり、仲間に声をかけたりする力がつく

- **カーマ博士**「大事なのは、遊び方。**夢中になれる力そのものは、きみの武器になる**んじゃよ。」
- **でい太**「夢中になる力を、他のことにも使えたらすごいことになりそう。でも……じゃあ、もうゲームはやっちゃだめってこと？」
- **カーマ博士**「そんなことはない。ゲームは楽しいし、力を育ててくれる、すばらしい道具じゃ。大事なのは、**わながあると知ったうえで、自分で選ぶ**ことじゃよ。」

（ヒントボックス・extra01/03の「抜け出すヒント」パターンを流用、`bg-green-50 border-green-200`）

1. 遊ぶ前に「何を・何分やるか」決める
2. 課金する前に24時間待ってみる（その気持ちが本当に大事かどうか、時間をおいて考える）
3. ランキングやフレンド機能の通知はオフにする
4. 「あと1回だけ」と思ったら、それがわなのサイン。ゲームのしくみを、自分の目標のために使ってみる

- **でい太の締め**（`bg-amber-50 border-2 border-amber-300` の箱で囲む）：「わなを知れば、かかりにくくなる。ゲームが悪いんじゃない。しくみを知った今のぼくたちは、遊ぶ側から、使いこなす側になれる。それが『考える力』だよ。」
- **ポン太**「カーマ博士、次はどんぐりゲーム作って。」

まとめボックス：ゲームには、記憶力・考える力・空間認識力・仲間と力を合わせる力を育てるはたらきがある。わなを知ったうえで、自分でどう付き合うか選べるようになることが大事。

### 場面5：自分の目標をゲーム化しよう

見出し：自分の目標をゲーム化しよう

- **でい太**「ゲームに夢中になれる力を、たとえばぼくがやってるサッカーの練習にも使えないかな。リフティングとか、なかなか続かなくて…。」
- **カーマ博士**「それ、できるんじゃよ。ゲームのしくみを、**自分の目標のために逆に使う**んじゃ。大人はこれを『ゲーミフィケーション』と呼んだりするが、むずかしい言葉は覚えなくていい。」
- **でい太**「サッカーの練習だと、具体的にはどうすればいいの？」
- **カーマ博士**「コツは3つじゃ。①大きな目標を**小さく区切る**。②今どこまで進んだかが**ひと目でわかるようにする**。③1つ終えるたびに、**すぐに達成感を感じられるようにする**。この3つがそろえば、ゲームと同じように『あと少し』でがんばれるようになるんじゃよ。『リフティング100回』を目指すなら、こんな感じじゃ。」

（応用例は4枚のカードに分けず、下記の1つの例（1枚のボックス）にまとめる。extra01のカードと同じ見た目でよいが、中に3つの要素を箇条書きで入れること）

⚽「リフティング100回チャレンジ日記」
- マイルストーンを5段階に区切る：10回→25回→50回→75回→100回。それぞれに「キック見習い」「リフティング少年」「リフティング中級」「リフティング上級」「リフティングマスター」という肩書きをつける
- 練習した日はカレンダーにスタンプを1個おす。その日「何回できたか」も数字で書きこむ（例：「7/1 ⚽ 5回」「7/5 ⚽ 12回」「7/10 ⚽ 25回 🎉［レベルアップ！］」）
- あとから記録を見返すと、「10日前はぜんぜんできなかったのに、今はこんなにできる」と、自分の成長がはっきりわかる

- **でい太**「サッカー以外にも使えるかな？　たとえば夏休みの宿題とか。」
- **カーマ博士**「もちろんじゃ。やり方はまったく同じじゃよ。」

（2つ目の応用例。ボックスは1つ目と同じ見た目で、中に3つの要素を箇条書きで入れる）

📚「夏休みの宿題プラン」
- 宿題をページ数で区切って、マイルストーンに肩書きをつける：「10ページ：ビギナー」→「30ページ：見習い卒業」→「50ページ：中級」→「80ページ：上級」→「全部：マスター」
- 宿題をやった日はカレンダーにスタンプを1個。スタンプが5個たまったらアイスを1個食べられる、のような小さなごほうびを先に決めておく
- 漢字ドリル・計算ドリル・読書感想文など、教科ごとに進捗バーを作って、終わった分だけ色をぬる

- **でい太**「これなら、8月31日にあわてて泣きながらやらなくてすみそう…！」
- **カーマ博士**「大きなゴールをひとつだけ置くんじゃない。途中に**小さなゴールをたくさん**置いて、進んだらすぐわかるようにする。サッカーでも、宿題でも、それが、ゲームのしくみを味方につけるコツなんじゃよ。」
- **でい太**「データ探検隊のミッションも、麻績村の謎を解くという大きなゴールを、レッスン1つ1つの小さなクリアに分けてるんだ。おんなじだね！」
- **ポン太**「オレも、どんぐり10個ごとに山にちいさい印をつけてる。」

まとめボックス（ページ全体の結論として少し大きめに）：ゲームのしくみを知れば、遊ばされる側から、自分の目標のために使う側になれる。大きな目標をひとつ置くだけでなく、①小さく区切る　②見える化する　③すぐに達成感を感じられるようにする。この3つを味方につけよう。

### Q1（3択・正解 b）

問題文：無料の宝箱やガチャで「あと1回だけ」と思ってしまうのはなぜ？

- a. 絶対に当たるように仕組まれているから
- b. 当たるかどうかがランダムで、いつ当たるかわからないから（**正解**）
- c. お金を返してもらえないから

正解時：⭕ せいかい！ いつ当たるかわからない「ランダムのごほうび」だからこそ、何度もやりたくなっちゃうんだ。お店のガチャガチャやカードのパック、大人のパチンコもぜんぶ同じしくみだよ。
不正解時：❌ もう一かい！ ヒント：カーマ博士は「当たるタイミングが決まっていたら、そんなに夢中にならない」って言ってたね。

### Q2（3択・正解 b）

問題文：毎日ログインボーナスがあったり、ランキングで友だちと比べさせたりするのはなぜ？

- a. プレイヤーが喜ぶ親切な機能だから
- b. 夢中にさせて「もっと強くなりたい」気持ちを大きくし、課金させるため（**正解**）
- c. ゲームのバグを直すため

正解時：⭕ せいかい！ 毎日遊んで夢中になるほど、「もっと強くなりたい」「負けたくない」気持ちが大きくなる。その気持ちが大きいときほど、よく考えられなくなって、つい課金してしまいやすくなるんだ。それが多くのゲーム会社の一番の収入源なんだよ。
不正解時：❌ もう一かい！ ヒント：ただ長く遊んでもらうだけでは、実はそんなに儲からないゲームも多いよ。遊んでいるうちに、どんな気持ちが大きくなっていくかを考えてみよう。
