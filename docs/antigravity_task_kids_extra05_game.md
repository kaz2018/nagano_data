# 実装依頼：番外編5「ゲームのわな」（kids/extra05）

## 概要

子どもコースの番外編5ページ目として、スマホゲーム・ソシャゲの「やめられないしくみ」を
会話形式で教えるページを新規作成する。番外編1「YouTubeのわな」・番外編3「インスタのわな」の
続編的な位置づけだが、本ページは「わな」の告発だけで終わらせず、**ゲームのいい面**と
**自分の目標への応用（ゲーミフィケーション）**まで踏み込む点が今までの番外編と異なる。

- **タイトル**: ゲームのわな
- **サブタイトル**: 「あと1回だけ」が、なぜやめられないの？
- **テーマカラー**: インディゴ（`indigo-500` / `indigo-600`）。body背景は `bg-indigo-50`
- **登場人物**: でい太・ポン太・**カーマ博士（新キャラ・ゲームクリエイター兼研究者）** の3人のみ

### キャラクター名の由来（背景情報）

「カーマ博士」は、**John Carmack（ジョン・カーマック）** へのオマージュとして命名。
id Softwareの共同創業者で、DOOM・Quakeの3D描画エンジンを自ら書き上げた伝説的プログラマー。
「天才プログラマー」の代名詞的存在であることから、オーナーの意向でこの人物にちなんだ名前を採用した。

- 実在の人物をそのまま模したキャラクターではなく、**名前の響きのみを引用したオリジナルキャラクター**である（ヒカケンジ兄さん／キラリ姉さんと同様の距離感）。人物の容姿・経歴を再現する必要はない。
- 見た目・性格設定はオリジナルで自由に決めてよい（本ドキュメントでは「ゲームクリエイター兼研究者」「白衣に眼鏡」程度の緩い方向性のみ指定）。
- 実在人物名（Carmack / カーマック）そのものは登場させない。あくまで「カーマ博士」という独立した名前として扱うこと。

## 会話スクリプト（確定稿・このまま使用すること）

セリフの文言は原則このまま使う。`**太字**` 部分は `<strong>` で強調する。

### 導入（タイトル直下のキャラ会話）

- **でい太**「最近、毎日学校から帰ったら真っ先にゲームのログインボーナスをチェックしちゃうんだ。忘れると『あー、もったいないことした』って気持ちになって。しかも、一度アプリを開くと、気づいたら1時間くらいたってる…。なんでこんなにやめられないんだろう。」
- **ポン太**「オレは毎朝、まず巣の外にどんぐりが増えてないか確認する。」
- **でい太**「そのしくみ、ちゃんとくわしい人に聞いてみたいな。カーマ博士に聞いてみよう。」

### 場面1：毎日ログインしたくなるわけ

見出し：毎日ログインしたくなるわけ

- **でい太**「カーマ博士！ 聞きたいことがあるんだ。毎日ログインボーナスをチェックしないと落ち着かない気持ちになるんだけど、これって何なの？」
- **カーマ博士**「正直に話そう。それは**わざと**そう感じるように作ってあるんじゃ。」

（仕掛けカード3枚・extra01の「ユーチューバーの工夫」パターンを流用）

- 🔥「連続ログインボーナス」：毎日開かないと記録が消えるから、休みたくても休めない気持ちにさせる
- 🏆「ランキング・フレンド機能」：友だちや知らない人と比べさせることで、負けたくない気持ちを利用する
- ⏳「終わらないクエスト・シーズンパス」：「あと少しで次のごほうび」を延々と用意して、やめる区切りをなくす

- **カーマ博士**「正直に言うとな、みんなに**長く・毎日**遊んでもらうほど、ゲーム会社は儲かる。だから、なるべく**やめるタイミングをなくす**ように工夫しているんじゃ。」
- **でい太**「YouTubeやインスタと同じだ…！ やめさせない工夫を、みんなしてるんだね。」
- **ポン太**「タヌキの世界にランキングがなくてよかった。」

まとめボックス：ログインボーナスもランキングも、「毎日・長く遊んでもらう」ためにわざと作られたしくみ。YouTubeやインスタと同じ「やめさせない工夫」。

### 場面2：一度始めるとやめられないわけ

見出し：一度始めるとやめられないわけ

- **でい太**「ログインしたあと、『ちょっとだけ』のつもりが、気づいたら1時間もたってるのはなぜなんだろう？」
- **カーマ博士**「それは、ゲームの中の『ごほうび』の出し方に秘密があるんじゃ。たとえば、無料でひける宝箱や、無料ガチャは、何が当たるか**完全にランダム**じゃろう？」
- **でい太**「うん、次はいいのが出るかもって、つい開けちゃう。」
- **カーマ博士**「そうじゃ。『次こそ当たるかも』と思わせることで、脳が興奮するように作られておる。これを**『可変報酬（かへんほうしゅう）』**と呼ぶ。もし当たるタイミングが決まっていたら、人はそんなに夢中にならない。**いつ当たるかわからない**からこそ、何度も引きたくなってしまうんじゃ。」
- **カーマ博士**「実はこれ、スロットマシンやくじ引きと**まったく同じしくみ**なんじゃよ。」
- **ポン太**「オレも『次のどんぐりは大きいかも』って思うと、つい掘っちゃう。」

まとめボックス：「あと1回だけ」でやめられないのは、ごほうびが当たるかどうかランダムだから。当たるタイミングがわからないからこそ、脳がもっと引きたくなるようにできている（可変報酬）。

### 場面3：ゲームが育てる力

見出し：ゲームが育てる力

- **でい太**「じゃあゲームって、悪いことしかないの…？」
- **カーマ博士**「そんなことはない！ わしはゲームを作る仕事に誇りを持っておる。ゲームには、きみの力を育てる、いいところもたくさんあるんじゃよ。」

（いい面カード4枚・extra03の「良いところ」ボックスパターンを流用、`bg-blue-50 border-blue-200`）

- 🧠「記憶力」：マップ・アイテム・手順を頭の中で整理して覚える力が鍛えられる
- 🧩「思考力・問題解決力」：パズルや戦略ゲームで「次の一手」を考える力がつく
- 🗺️「空間認識力」：3Dゲームの世界で位置や距離をつかむ力が育つ
- 🤝「協調性」：チームプレイで役割を分担したり、仲間に声をかけたりする力がつく

- **カーマ博士**「大事なのは、遊び方。**夢中になれる力そのものは、きみの武器になる**んじゃよ。」
- **でい太**「夢中になる力を、他のことにも使えたらすごいことになりそう。」

まとめボックス：ゲームには記憶力・思考力・空間認識力・協調性を育てる力がある。夢中になる力そのものは、使い方しだいで武器になる。

### 場面4：自分の目標をゲーム化しよう

見出し：自分の目標をゲーム化しよう

- **でい太**「ゲームに夢中になれる力を、勉強とか他のことにも使えないかな。」
- **カーマ博士**「それ、できるんじゃよ。ゲームのしくみを、**自分の目標のために逆に使う**んじゃ。これを**『ゲーミフィケーション』**という。」

（応用例カード4枚・extra01の「1年後どうなってるか」グリッドパターンを流用）

- 📖「読書クエスト」：読んだページ数で自分をレベルアップ。10冊読んだら『読書レベル5』達成！
- ⛷️「連続記録（ストリーク）」：毎日ちょっとでも練習した日をカレンダーに記録。連続記録が途切れないようにがんばる
- 📝「宿題クエスト」：「今日のクエスト：宿題を終わらせる」のように、やることをクエスト化して「クリア」の達成感を味わう
- 🏅「じぶん実績」：「毎日5分勉強」「1週間休まず練習」など、自分だけの「実績」を作ってクリアするたびに自分をほめる

- **カーマ博士**「連続記録もランキングも、ゲーム会社が使っているのと同じしくみじゃ。でも、それを**自分をあやつる側**で使えば、こんどはきみの味方になる。」
- **でい太**「よし！ データ探検隊のミッションも、麻績村の謎を解く『レベルアップクエスト』だと思ってがんばるぞ！」
- **ポン太**「オレも『どんぐり100個クエスト』やってる。」

まとめボックス：レベル・クエスト・連続記録は、自分の目標のためにも使える。ゲーム会社に使われる側から、自分で使う側になろう。

### 場面5：わなを知れば、自分で選べる

見出し：わなを知れば、自分で選べる

- **でい太**「じゃあ、もうゲームはやっちゃだめってこと？」
- **カーマ博士**「そんなことはない。ゲームは楽しいし、力を育ててくれる、すばらしい道具じゃ。大事なのは、**わながあると知ったうえで、自分で選ぶ**ことじゃよ。」

（ヒントボックス・extra01/03の「抜け出すヒント」パターンを流用、`bg-green-50 border-green-200`）

1. 遊ぶ前に「何を・何分やるか」決める
2. 課金する前に24時間待ってみる（その気持ちが本当に大事かどうか、時間をおいて考える）
3. ランキングやフレンド機能の通知はオフにする
4. 「あと1回だけ」と思ったら、それがわなのサイン。ゲームのしくみを、自分の目標のために使ってみる

- **でい太の締め**（`bg-amber-50 border-2 border-amber-300` の箱で囲む）：「わなを知れば、かかりにくくなる。ゲームが悪いんじゃない。しくみを知った今のぼくたちは、遊ぶ側から、使いこなす側になれる。それが『考える力』だよ。」
- **ポン太**「カーマ博士、次はどんぐりゲーム作って。」

まとめボックス（ページ全体の結論として少し大きめに）：ゲームのしくみを知れば、遊ばされる側から、自分の目標のために使う側になれる。

## やってみよう（クイズ仕様）

実装方式は extra01 と同じ：ラジオボタン3択＋1問ごとの「答え合わせ」ボタン、
結果は `hidden` 解除で表示。ボタン色は `bg-indigo-500 hover:bg-indigo-600`。
正解フィードバックは `bg-green-50 text-green-700 border-green-200`、
不正解は `bg-red-50 text-red-600 border-red-200`（extra01のJSパターンを流用し関数名も同様に checkQ1〜, showQ4Answer）。

### Q1（3択・正解 b）

問題文：無料の宝箱やガチャで「あと1回だけ」と思ってしまうのはなぜ？

- a. 絶対に当たるように仕組まれているから
- b. 当たるかどうかがランダムで、いつ当たるかわからないから（**正解**）
- c. お金を返してもらえないから

正解時：⭕ せいかい！ いつ当たるかわからない「ランダムのごほうび」だからこそ、脳がもっと引きたくなっちゃうんだ。これを「可変報酬」って言うよ。
不正解時：❌ もう一かい！ ヒント：カーマ博士は「当たるタイミングが決まっていたら、そんなに夢中にならない」って言ってたね。

### Q2（3択・正解 b）

問題文：毎日ログインボーナスがあったり、ランキングで友だちと比べさせたりするのはなぜ？

- a. プレイヤーが喜ぶ親切な機能だから
- b. できるだけ毎日・長く遊んでもらうため（**正解**）
- c. ゲームのバグを直すため

正解時：⭕ せいかい！ 毎日・長く遊んでもらうほど、ゲーム会社は儲かるしくみになっているんだ。
不正解時：❌ もう一かい！ ヒント：YouTubeやインスタの「わな」と同じで、やめるタイミングをなくす工夫だったね。

### Q3（自由記述・模範解答表示式）

問題文：ゲームで身についた（身につきそうな）力があるとしたら、何だと思う？

選択肢・入力欄なし。「こたえのれいを見る」ボタン（`showQ3Answer()`）で模範解答を表示：

> （れい）マップを覚える記憶力、次の一手を考える思考力、チームで声をかけあう協調性など、遊びながら育つ力はたくさんあるよ。

### Q4（自由記述・模範解答表示式）

問題文：自分の目標を1つ、クエストや実績みたいに書いてみよう。

選択肢・入力欄なし。「こたえのれいを見る」ボタン（`showQ4Answer()`）で模範解答を表示：

> （れい）「読書レベル5（10冊読む）」「毎日5分勉強クエスト」「1週間休まず練習の実績」など。ポイントは、目標を「小さく区切って」「達成感が見える形」にすることだよ。

## 保護者の方へ（セクション内容）

> このページでは、スマホゲーム・ソシャゲに使われている「可変報酬（ランダムなごほうびによる依存性）」「連続ログインボーナス」「終わりのない設計」など、ゲーム業界で実際に使われている設計手法を子ども向けに解説しました。同時に、記憶力・思考力・空間認識力・協調性など、ゲームが育てる力があることや、ゲームのしくみを自分の目標達成（ゲーミフィケーション）に応用する方法もあわせて伝えています。
