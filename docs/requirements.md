# 要件定義書 - 長野県データ分析教育サイト

- **目的**: 長野県の人口データを使い、地元の子供・大人がデータ分析を学べる教育サイト
- **主な学習者**: 麻績村・筑北村の小学生
- **対象**: 長野県内の小学生〜大人
- **言語**: 日本語のみ
- **デプロイ先**: GitHub Pages（https://kaz2018.github.io/nagano_data/）
- **更新日**: 2026-03-19（会話セッション5終了時点）

---

## ターゲット別要件

### フェーズ1：子ども向け（小学生）

| 項目 | 内容 |
|------|------|
| ツール | Google Sheets（リンク提供） |
| データ | 長野県毎月人口異動調査の実データ（市町村レベル・9年分） |
| 内容 | 棒グラフ・折れ線グラフの作成、並べ替え、地域比較 |
| 目標 | 「データで何かわかる！」という体験を得る |

### フェーズ2：大人向け

| 項目 | 内容 |
|------|------|
| ターゲット | 麻績村・筑北村の子どもの保護者を中心とした大人 |
| ユーザー数 | 少数（数十人規模を想定） |
| スキル | ばらつきが大きい（Excel未経験〜ある程度使える人まで混在） |
| 文章レベル | 中学2〜3年生が読める内容。短文・改行多め・平易な日本語 |
| ツール | Google Sheets → Looker Studio → Python（段階的） |
| データ | 長野県毎月人口異動調査の元Excelファイル（生データ） |
| 目標 | 「データを整える→集計する→見せる」の流れを体験する |

#### レッスン構成

| レッスン | レベル | テーマ | 内容 |
|---------|--------|--------|------|
| L1 | 初級 | データクレンジング | 神Excelの問題点（全角数字・空白行・小計行・メモ） |
| L2 | 初級 | pivot集計 | きれいなデータをピボットテーブルで集計する。後半で表記ゆれ・IDの活用を扱う |
| L3 | 中級 | Looker Studio | Sheetsのデータをダッシュボードで可視化・共有する |
| L4 | 中級 | Python（pandas）| Google Colabでpandasを使ったデータ整形と可視化 |

#### L2 後半で扱う「表記ゆれ・ID」の概念

- 人名の姓と名の間に空白があると、同じ人が別人に見えてしまう
- 部署名・商品名などが微妙に違う表記（「営業部」「営業１部」）だと集計できない
- こうした問題を防ぐために「ID（番号）」で管理する考え方を紹介する

## 開発規約

### Python 実行

Python は必ず `uv run python3` で実行する（`.venv/bin/python3` や `python3` 直接呼び出しは禁止）。

```bash
uv run python3 script.py
uv run python3 -c "..."
```

### CSVの読み込み

CSVファイルはすべて BOM 付き UTF-8 (`utf-8-sig`) で作成・読み込みを行う。

## UIデザイン規約（子ども向けレッスン）

### 答え合わせのパターン

- 「やってみよう」セクションの各問には、**問ごとに「答え合わせ」ボタン**を設ける
- ボタン押下で、その問の直下に結果を表示する（⭕ せいかい！ / ❌ もう一かい！）
- 全問まとめて確認するボタンやスコア表示は設けない
- **自由記述・理由を問う問題**（正解が一つでない問題）は正誤判定せず、「こたえのれいを見る」ボタンで模範解答を表示する
- 部分一致が想定される問（例：3つ選ぶ問題）は、何個正解かを △ で表示する

## データ仕様

- **出典**: 長野県毎月人口異動調査（住民基本台帳ベース、各年1月1日現在）
- **収録年**: 2017〜2025年（9年分、各年1月）
- **市町村数**: 77市町村（19市・23町・35村）
- **項目**: 市町村名・総人口・男・女
- **注意**: 国勢調査確定値とは異なる推計値
